# portfolio-simulator

### Arguments:
Given _n_ assets
- **timeframe**: *t* el of [w, 2w, m],
- **day**: *d* el of [1...l], where _l < timeframe_,
- **portfolio** (best assets): _p <= n_,
- **cutoff**: _p <= c <= n_,
- **selector**: [1...l], where _l < timeframe_,
- **begin**: beginning of the period,
- **end**: end of the period
